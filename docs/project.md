# О проекте ZoneRec

ZoneRec — небольшая утилита для Linux/X11, которая записывает выбранную область
экрана в MP4 и сохраняет снимки экрана. Приложение работает из системного трея,
поддерживает глобальные горячие клавиши и отдельные CLI-команды.

## Возможности

- выбор прямоугольной области мышью;
- свободное или фиксированное соотношение сторон;
- запись H.264/MP4 через `ffmpeg` и `x11grab`;
- захват микрофона, системного звука или обоих источников;
- красная рамка вокруг записываемой области;
- PNG/JPEG-снимки с копированием в буфер обмена;
- настройки каталогов, FPS, пресета кодирования и горячих клавиш.

## Границы поддержки

ZoneRec рассчитан на X11. Нативный захват Wayland через PipeWire и порталы пока
не реализован. Основная целевая среда — Debian/Ubuntu и производные дистрибутивы
с GTK 3 и PulseAudio-совместимым звуковым сервером.

## Основные файлы

- `zonerec.py` — приложение, интерфейс, захват и CLI;
- `zonerec.desktop` — ярлык и автозапуск;
- `install.sh` — пользовательская установка в `~/.local`;
- `build-deb.sh` — сборка Debian-пакета;
- `assets/zonerec.svg` — иконка приложения;
- `README.md` — пользовательская документация.

## Пользовательские данные

- конфигурация: `${XDG_CONFIG_HOME:-~/.config}/zonerec/config.ini`;
- состояние активной записи: `${XDG_CACHE_HOME:-~/.cache}/zonerec/state`;
- записи и снимки: каталоги из `config.ini`.
