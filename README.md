**ZoneRec** — это легковесное приложение, которое живёт в системном трее и позволяет мгновенно выбирать область экрана мышкой и записывать её в формате MP4 по горячей клавише. Во время записи выбранная зона подсвечивается аккуратной **красной рамкой** (которая не попадает в итоговое видео!).

## ✨ Ключевые возможности

- 🎯 **Выделение мышкой**: Рисуйте прямоугольник в любой части экрана (с поддержкой фиксированных пропорций: `16:9`, `1:1`, `4:3` и др.).
- 🎥 **Красный индикатор**: Во время записи вокруг снимаемой области появляется красная рамка-оверлей, чтобы вы всегда знали, что попадает в кадр.
- ⚡ **Глобальные хоткеи**: Начинайте и останавливайте запись, а также делайте мгновенные скриншоты по горячим клавишам из любого окна.
- 📸 **Скриншоты**: Сохранение выделенной зоны в `PNG` или `JPEG`, с возможностью поправить рамку перед сохранением и автоматическим копированием в буфер обмена.
- 🎛️ **Удобный трей**: Настройки, папки сохранения, выбор источников звука и пресетов — всё в простом графическом меню.
- 🎧 **Аудио-режимы**: Захватывайте системный звук, микрофон, и то и другое вместе, или пишите без звука.

## 🚀 Установка

ZoneRec легко установить в Ubuntu, Debian, Linux Mint и других DEB-совместимых дистрибутивах.

### Вариант 1: Установка готового `.deb` (Рекомендуется)

1. Перейдите на страницу **[Releases](https://github.com/ar-webcetera/zonerec/releases/latest)** и скачайте актуальный файл `zonerec_*_all.deb`.
2. Установите пакет в терминале:
   ```bash
   sudo apt install ./zonerec_0.1.11_all.deb
   ```
3. После установки найдите **ZoneRec** в меню приложений вашей системы.

### Вариант 2: Сборка `.deb` пакета вручную

```bash
git clone https://github.com/ar-webcetera/zonerec.git
cd zonerec
./build-deb.sh
sudo apt install ./dist/zonerec_*_all.deb
```

## ⌨️ Горячие клавиши (по умолчанию)

Вы можете изменить любые комбинации в **Настройках** приложения.

| Действие | Комбинация |
| :--- | :--- |
| **Выбрать область и начать запись** | <kbd>Super</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd> |
| **Остановить запись** | <kbd>Super</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> |
| **Сделать скриншот области** | <kbd>Super</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> |

## ⚙️ Использование через терминал (CLI)

Вы можете автоматизировать ZoneRec или привязать его к своим системным хоткеям, используя команды CLI:

```bash
zonerec             # Запустить трей-демон (работает в фоне)
zonerec --start     # Выбрать область мышкой и сразу начать запись
zonerec --stop      # Остановить текущую запись
zonerec --toggle    # Запустить запись, если остановлена, или остановить, если идёт
zonerec --shot      # Выбрать область мышкой, при необходимости поправить рамку и сделать скриншот
zonerec --settings  # Открыть графическое окно настроек
```

## 🛠 Зависимости

Если вы используете скрипт установки или `.deb` пакет, все зависимости подтянутся **автоматически**:
- `ffmpeg` (для записи экрана `x11grab`)
- `python3-gi`, `gir1.2-gtk-3.0`, `gir1.2-gdkpixbuf-2.0` (интерфейс и буфер обмена)
- `gir1.2-keybinder-3.0` (глобальные горячие клавиши)
- `libnotify-bin`, `gir1.2-notify-0.7` (системные уведомления)
- `pulseaudio-utils` (поиск микрофонов и системных звуковых карт)

> ⚠️ **Важно:** ZoneRec работает **только в X11-сессии**. Поддержка Wayland в данный момент не реализована (на Wayland для захвата требуются другие инструменты, такие как `slurp` + `wl-screenrec`).
