# ZoneRec

ZoneRec — минималистичная утилита для Linux/X11: выбрать область экрана по горячей клавише и записать её в MP4. Приложение живёт в трее, управляется глобальными хоткеями и имеет графические настройки.

Под капотом используется `ffmpeg` (`x11grab`) для записи и кодирования. Выбор области встроен в приложение.

## Статус

- Поддерживается: Linux/X11.
- Не поддерживается: Wayland.
- Основной формат установки: `.deb` для Debian/Ubuntu/Linux Mint и совместимых дистрибутивов.

## Возможности

- Выбор области экрана мышкой и старт записи одной горячей клавишей.
- Корректная остановка записи с финализацией MP4.
- Трей-меню: запись, скриншот, стоп, папка записей, настройки, выход.
- Глобальные хоткеи через Keybinder.
- Настройки: хоткеи, папки сохранения, пропорции области, FPS, пресет ffmpeg, источники аудио.
- Свободное выделение или популярные пропорции: `16:9`, `9:16`, `4:3`, `3:4`, `1:1`, `21:9`.
- Скриншоты выбранной области в PNG или JPEG.
- Запись без звука, с микрофоном, со звуком системы или с обоими источниками.
- Уведомления о старте и остановке записи.
- Автозапуск при входе в систему.

## Установка

### Готовый `.deb`

Скачайте `.deb` из GitHub Releases, перейдите в папку загрузки и установите:

```bash
sudo apt install ./zonerec_0.1.10_all.deb
```

После установки ZoneRec появится в меню приложений и будет запускаться через иконку.

### Сборка `.deb` из исходников

Если пакет собран локально, он появится в `dist/`:

```bash
./build-deb.sh
sudo apt install ./dist/zonerec_0.1.10_all.deb
```

### Локальная установка из исходников

В уже склонированном репозитории:

```bash
./install.sh
zonerec
```

Скрипт установит приложение в `~/.local/bin`, ярлык в меню приложений и автозапуск в `~/.config/autostart`.

## Зависимости

Установочные скрипты подтягивают зависимости автоматически:

- `ffmpeg`

- `xdg-utils`
- `pulseaudio-utils`
- `gir1.2-appindicator3-0.1` или `gir1.2-ayatanaappindicator3-0.1`

## Запуск

```bash
zonerec             # трей-демон
zonerec --start     # выбрать область и начать запись
zonerec --stop      # остановить запись
zonerec --toggle    # старт/стоп
zonerec --screenshot # скриншот выбранной области
zonerec --settings  # открыть настройки
```

## Хоткеи

По умолчанию:

| Действие | Клавиши |
|---|---|
| Выбрать область и начать запись | `Super+Shift+R` |
| Остановить запись | `Super+Shift+S` |
| Скриншот выбранной области | `Super+Shift+P` |

Изменить хоткеи можно в меню трея: **Настройки**.

## Область и скриншоты

В настройках можно выбрать свободное выделение или фиксированную пропорцию области. При фиксированной пропорции рамка во время выделения сразу держит выбранный формат.

Скриншоты сохраняются в отдельную папку, по умолчанию:

```text
~/Videos/ZoneRec/Screenshots
```

Фактическая папка видео берётся из XDG-настроек системы, поэтому в локализованных окружениях это может быть, например, `~/Видео/ZoneRec/Screenshots`.

## Аудио

В настройках можно выбрать режим записи:

- `Без звука`
- `Микрофон`
- `Звук системы`
- `Микрофон + звук системы`

Список доступных PulseAudio-источников можно посмотреть вручную:

```bash
pactl list sources short
```

Для записи звука системы обычно нужен источник с суффиксом `.monitor`.

## Конфиг

Настройки хранятся в `~/.config/zonerec/config.ini`:

```ini
[zonerec]
hotkey_select_start = <Super><Shift>r
hotkey_stop = <Super><Shift>s
hotkey_screenshot = <Super><Shift>p
output_dir = /home/you/Videos/ZoneRec
screenshot_dir = /home/you/Videos/ZoneRec/Screenshots
screenshot_format = png
fps = 30
preset = veryfast
selection_aspect = free
audio = none
audio_mic =
audio_system =
```

Обычно редактировать файл вручную не нужно: всё основное доступно в графических настройках.

## Удаление

Если приложение установлено из `.deb`:

```bash
sudo apt remove zonerec
```

Если приложение установлено через `./install.sh`, удалите пользовательские файлы:

```bash
rm -f ~/.local/bin/zonerec
rm -f ~/.local/share/applications/zonerec.desktop
rm -f ~/.config/autostart/zonerec.desktop
rm -f ~/.local/share/icons/hicolor/scalable/apps/zonerec.svg
rm -f ~/.local/share/pixmaps/zonerec.svg
```

Пользовательский конфиг и записи не удаляются автоматически.

## Ограничения

ZoneRec работает только в X11-сессии. Для Wayland нужен другой бэкенд, например `slurp` + `wf-recorder`/`wl-screenrec` через PipeWire-портал. В текущую версию это не входит.
